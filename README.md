#  Klovy Chat — Strona Informacyjna

Statyczna strona informacyjna komunikatora **Klovy Chat** — prezentuje funkcje, zasady korzystania oraz informacje dla użytkowników platformy **Klovy Systems**.

---

##  O projekcie

Klovy Chat Information to prosta strona statyczna stanowiąca publiczne źródło informacji o komunikatorze Klovy Chat.

---

##  Uruchomienie lokalne

```bash
# Sklonuj repozytorium
git clone https://github.com/klovy-systems/klovy-chat-information.git
cd klovy-chat-information
```

Otwórz `index.html` bezpośrednio w przeglądarce lub skorzystaj z rozszerzenia **Live Server** w VS Code dla wygodniejszego podglądu na żywo.
